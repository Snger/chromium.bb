This is sample fileHandler extension that uses NaCl plugin to read .txt files on ChromiumOS. To run the extension, please open File Manager (ctrl + M), select some .txt file and click button "Run file-io test extension". As a result, extension will output the selected file's content or error message (if extension fails during reading the file, error message will be the same as the error code). One instance of the extension will be able to open a file only once.
